Tutorial of ABDJ

Tutorial

As soon as you want to use this product, make sure your DAW supports the OSC protocol.

Get ABDJ

  1. Download ABDJ on Google Play.
  2. Make sure your computer, which you are working with and your phone, where you have installed the ABDJ app, are on THE SAME network (eg. LAN, Wi-Fi...). Find its IP address.
  3. Click the button "Get Plugin" and the ABDJ Plugin will be installed in your computer.

Warning: Especially on Macs, there is no guarantee that ABDJ Plugin will be easily installed. Make sure you have changed permissions to install things to computer from another devices.

Use in practice

  1. Import an audio track to your DAW, enable and open ABDJ.
  2. Click the button "Get IP Address (Your Computer)", it will return your computer's IP address.
  3. Open the ABDJ app, write computer's IP address and click the button "Connect".
  4. If it is alright, you should hear changes of sound when manipulating with the app.

Happy mixing!
