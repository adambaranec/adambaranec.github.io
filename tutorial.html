Tutorial

As soon as you would like to use this product, make sure your DAW supports OSC protocol.

Get OscDJ

  1. Download OscDJ on Google Play. You will also get the OscDJ Plugin (VST3) as a package for Windows, Mac OS and Linux.
  2. Install the plugin (named OscDJ) to your DAW.
  3. Make sure your computer, which you are working with and your phone, where you have installed the OscDJ app, are on THE SAME network (eg. LAN, Wi-Fi...).

Use in practice

  1. Import an audio track to your DAW, enable and open OscDJ.
  2. Click the button "Get IP Address (Your Computer)", it will return your computer's IP address.
  3. Open the OscDJ app, write computer's IP address and click the button "Connect".
  4. If it is alright, you should hear changes of sound.

Happy mixing!
