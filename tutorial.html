Tutorial

As soon as you want to use this product, make sure your DAW supports the OSC protocol.

Get OscDJ

  1. Download OscDJ on Google Play.
  2. Make sure your computer, which you are working with and your phone, where you have installed the OscDJ app, are on THE SAME network (eg. LAN, Wi-Fi...) and find its IP address.
  3. Click the button "Get Plugin" and the OscDJ Plugin will be installed in your computer.

Use in practice

  1. Import an audio track to your DAW, enable and open OscDJ.
  2. Click the button "Get IP Address (Your Computer)", it will return your computer's IP address.
  3. Open the OscDJ app, write computer's IP address and click the button "Connect".
  4. If it is alright, you should hear changes of sound.

Happy mixing!
