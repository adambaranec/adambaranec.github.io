NetDJ

Welcome to the NetDJ project

How it works

This project contains two types: App and Plugin. The App is practically a DJ console in the digital form.
It controls the Plugin, which has to be installed in a computer with the help of the protocol OSC - open sound control.
It means that in the App, the IP address of a computer, where the Plugin will be controlled, must be defined.

Both mobile (App) and computer (Plugin) have to be connected to the same Wi-Fi network!

Warning: This project is under construction at the moment. Although it has been not tested
at some events yet, it is possible that it does not have to work if a Wi-Fi network has poor quality.

Tutorial

1. Buy App (after you will receive the Plugin to your e-mail address to install).

2. In the App, write to the field on the right top IP address of computer, where the Plugin is installed.

3. Open Plugin and choose an audio file to play (.mp3, .wav, .aiff etc.).

Happy jamming!
