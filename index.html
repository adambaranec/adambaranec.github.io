OscDJ

Welcome to the OscDJ official page

What is it about?

OscDJ serves as a DJ console in a mobile, which is connected to a computer wirelessly.

How does it work?

This project works with a protocol called OSC - open sound control, capable for sending data from an application to another application within a network.
